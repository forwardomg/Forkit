Сервисы

  • CanIUse - Сведения о текущем состоянии поддержки браузерами возможностей HTML5.
  • Modernizr - Библиотека Javascript для определения возможностей и управления случаями восстановления HTML5.
  • html5boilerplate - Набор шаблонов HTML, CSS и JS с оптимальными алгоритмами для разработки кода HTML5 и внешнего интерфейса.
  • HTML5 Cross-browser Polyfills - Набор скриптов, которые активируют функции HTML5 во всех браузерах.
  • Let's make the web faster - Полный список инструментов, связанных с производительностью веб-приложений.
